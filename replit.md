# FunPay Cardinal Bot

## Огляд проекту

FunPay Cardinal - це бот для автоматизації роботи з платформою FunPay (біржа ігрових товарів та послуг). Проект включає:

- Автоматичну відповідь на повідомлення
- Автоматичну доставку товарів
- Інтеграцію з Telegram для керування ботом
- Систему плагінів для розширення функціональності
- Мультимовну підтримку (українська, російська, англійська)
- HTTP Keep-Alive сервер для запобігання відключення

## Налаштування

### Перше налаштування

При першому запуску бот автоматично запустить майстер налаштування `first_setup.py`, який попросить вас ввести:

1. **Golden Key (FunPay)** - ключ авторизації FunPay
2. **Telegram Bot Token** - токен вашого Telegram бота (отримайте через @BotFather)
3. **Telegram User ID** - ваш Telegram ID для адміністрування

### Отримання Golden Key

1. Увійдіть на FunPay через браузер
2. Відкрийте інструменти розробника (F12)
3. Знайдіть cookie `golden_key`
4. Скопіюйте його значення

### Створення Telegram бота

1. Напишіть @BotFather в Telegram
2. Виконайте команду `/newbot`
3. Дотримуйтесь інструкцій
4. Скопіюйте отриманий токен

### Отримання Telegram ID

1. Напишіть @userinfobot в Telegram
2. Отримайте свій ID

## HTTP Keep-Alive сервер

Бот включає HTTP сервер для запобігання автоматичного відключення та моніторингу:

- **Порт:** 5000
- **Адреса:** http://localhost:5000
- **Ендпоінти:**
  - `GET /` - перевірка статусу бота
  - `GET /status` - детальна інформація про статус
  - `GET /ping` - простий пінг

### Важливо: UptimeRobot

⚠️ **UptimeRobot офіційно заблокирован Replit з 2024 року**

На жаль, UptimeRobot більше не працює з Replit через блокування Replit и зміну DNS. Однак у вас є альтернативи:

### Рекомендовані рішення для 24/7 роботи:

1. **Replit Core Subscription** ($20/місяць)
   - Включає $25 в кредитах на развертывання
   - Развертуйте бота как "Autoscale" deployment
   - Бот буде arbeiten 24/7 без перерв

2. **Інші сервіси моніторингу:**
   - Ваш HTTP сервер дозволяє використовувати інші сервіси (UptimeFLare, StatusCake тощо)
   - Скористайтесь URL: `https://your-replit-app.replit.dev/status`

3. **Тимчасове рішення:**
   - Вільний рівень Replit дозволяє ботові працювати протягом деякого часу
   - Для найменшої затримки увімкніть автоподнятие в конфігурації бота
